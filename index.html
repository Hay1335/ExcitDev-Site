Server Rules

  • #1No griefing, stealing, or destroying other players' builds or property.
  • #2No hacking, cheating, or using unauthorized mods that give unfair advantages.
  • #3Be respectful in chat. No harassment, hate speech, or excessive profanity.
  • #4No spamming, advertising other servers, or posting inappropriate content.
  • #5No exploiting bugs or glitches. Report them to staff immediately.
  • #6Listen to staff. Their decisions are final. Disputes go through the help desk.

Frequently Asked Questions

The server runs the latest stable Java Edition release. We update within a few weeks of each major Minecraft version to ensure plugin compatibility.

Use a golden shovel to mark two corners of your claim area. You start with 500 claim blocks and earn more by playing. Type /claimslist to see your claims.
If no Minecraft server is running at excitdev.org yet, it will show as offline. Check our Discord for announcements about launch dates and maintenance windows.
Use the contact form below and select "Staff Application" as the topic. Include your age, time zone, past moderation experience, and how many hours a week you can commit.
Visit map.excitdev.org to see FluffyMap — a live, interactive overhead map of the entire server world. You can see player locations, claimed land, and biomes.
Optifine, Sodium, and approved performance mods are fine. X-ray, auto-clickers, and any mods that provide unfair game advantages are strictly banned.
